CSS Color Cheatsheet

Color Attributes

Attribute Name Description Example
color: Keyword color Named color of display text. color: 'green'
rgb() Red, Blue, Green Numeric value mix of red, blue, green. Value from 0 to 255. rga(0,255, 123)
hsl() Hue-Saturation-Light Numeric values of Hue-Saturation-Light.Valuse from 0 to 360. hsl(120, 60%, 70%)
# Hexademical Hex color # followed by 3 or 6 characters. Value for red, blue and green. 16 digits; 0 to 9 followed by "A" to "F". 6 value hex can be shortened to 3 digits. #FF0000
hsla Opacity in hsl Opacity added to hsl(). alpha is a decimal between 0 and 1. hsla(34, 100%, 50%, 0.1)
rgba Opacity in rgba Like hsla, opacity is added to rgb(). rgba(234, 45, 98, 0.33)

Key points to remember

  • color: Foreground color i.e. color of text
  • back-ground color: Background color of the element
  • Named color: English Words to descrobe color also called "Keyword Colors"
  • Transparent: Zero Opacity, used like a color keyword
